docker-compose 
搭建二次元和独角兽卡网


修改 
```nginx/conf.d/*.conf ```
中的域名为自己的。

然后 运行 
```_start.sh```
脚本 即可自动部署。


在 nginx/docker-compose.yaml 中注释掉不搭建的其中之一.

##  安装提醒

mysql redis 服务器地址 填 host.docker.internal


##  环境准备
debian系统
```bash
sudo apt update && sudo apt -y upgrade && sudo apt install docker docker.io docker-compose
```
